albania-city-api

Star Issue Follow @911mateo011

An updated database with all the cities of albania as of 2021.

Get information about cities and regions of albania

Example:

https://albanian-city-api.herokuapp.com/city?q=ballsh

Usage

This api responds with Json format.It's hosted on firebase and uses cloud firestore as a database.

Endpoints

Get all cities

GET https://albanian-city-api.herokuapp.com/cities

Response as JSON:

[{
     "name": "Bajram Curri",
     "district": "Tropoje",
     "region": "Kukes"
    },
    {
     "region": "Shkoder",
     "name": "Bajze",
     "district": "Malesi e madhe"
    },
    {
     "district": "Mallakaster",
     "region": "Fier",
     "name": "Ballsh"
    }]

Get one city

GET https://albanian-city-api.herokuapp.com/city?q={city_name}

Response as JSON:

{
 "region": "Fier",
 "name": "Ballsh",
 "district": "Mallakaster"
}
